EV Website
Project Overview

This project is an Electric Vehicle (EV) website that showcases information about various EV models, charging stations, and EV news. It is designed to be interactive, responsive, and user-friendly for visitors interested in electric vehicles.

Features
Home page with interactive hero section
EV model listing with images and specifications
Charging station locator
News and updates about electric vehicles
Contact form for inquiries
Fully responsive design for desktop and mobile
Technologies Used
Frontend: HTML, CSS, JavaScript, Bootstrap, React

Installation

Clone the repository:

git clone <your-repo-link>

Navigate to the project folder:

cd ev-website

Install dependencies:

npm install

Start the development server:

npm start
Usage
Open the website in your browser
Browse through EV models and charging stations
Use the contact form to send inquiries
